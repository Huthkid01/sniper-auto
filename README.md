# Sniper Autos - Premium Car Rental & VIP Security

Sniper Autos is a premium car rental and VIP security service website designed for discerning travelers who demand luxury, comfort, and safety. This project showcases a high-end fleet of SUVs and comprehensive security solutions.

## 🚀 Features

-   **Premium Fleet Showcase**: Detailed listings of luxury vehicles including Mercedes G-Wagon, Lexus LX570, Range Rover, and more.
-   **VIP Security Services**: Dedicated section for Executive Protection, Secure Transport, and Event Security.
-   **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
-   **Interactive Booking System**:
    -   User-friendly form with vehicle and service selection.
    -   Custom date picker using [Flatpickr](https://flatpickr.js.org/).
    -   Stylish custom notification popups upon submission.
-   **Theme**: Elegant "Gold & Dark Slate" color scheme reflecting luxury and prestige.
-   **Contact Integration**: Direct WhatsApp links and contact information.

## 🛠️ Technologies Used

-   **HTML5**: Semantic structure and layout.
-   **CSS3**: Custom styling, Flexbox/Grid layouts, and responsive media queries.
-   **JavaScript (Vanilla)**: DOM manipulation, form handling, and interactive elements.
-   **Flatpickr**: Lightweight and powerful datetime picker.
-   **Font Awesome**: Scalable vector icons.
-   **Google Fonts**: Montserrat and Open Sans typography.

## 🔧 Setup & Usage

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/sniper-autos.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd sniper-autos
    ```
3.  Open `index.html` in your preferred web browser to view the site.

## 📱 Mobile Responsiveness

The site is built with a "Mobile-First" approach where applicable, ensuring:
-   Stacked layouts for smaller screens.
-   Touch-friendly buttons and inputs.
-   Hidden/Adaptive elements for better mobile UX.
